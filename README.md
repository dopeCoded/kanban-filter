# kanban
 Trelloにカレンダー機能をつけたものです。<br >
 タスクに締切日をつけるとカレンダーに同期されるため、いつまでに終わらせるタスクなのかをみることができます。 <br >
 尚、Doneに入っているタスクについてはカレンダーに同期されません。
